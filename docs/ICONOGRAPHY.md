# Iconographie

## Référentiel

Tous les pictos du site (sidebar, sections, éléments, section « Nos services », landing) s’appuient sur le **OVHcloud Design System – Iconography** :

**https://zeroheight.com/6fc8a63f7/p/49e861-iconography**

Les fichiers SVG utilisés dans l’app se trouvent dans `public/pictos/` et sont alignés sur ce design system. La source unique des chemins d’icônes est `lib/pictos.ts` ; la section « Nos services » utilise la même source via `lib/service-pictos.ts`.

## Image d’onboarding

L’illustration de la page d’accueil / onboarding (hero) est fournie au format PNG dans `public/hosting-hero.png` (sécurité, clavier, HTTPS, thème professionnel).
